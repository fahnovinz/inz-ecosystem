# Contributing to claude-oss-kit

Thank you for helping improve this toolkit for open-source contributors!

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test locally:
   ```bash
   node bin/claude-oss-kit.js check YOUR_USERNAME
   ```
5. Commit with a clear message
6. Open a Pull Request

## Good First Issues

- Add npm/PyPI download count checking
- Improve error messages for rate limiting
- Add Indonesian translations for CLI output
- Write tests for eligibility logic
- Add support for GitLab contribution checking

## Code Style

- Keep dependencies minimal (zero runtime deps preferred)
- Use clear variable names
- Match existing file structure

## Pull Request Guidelines

- One feature/fix per PR
- Update README if adding new commands
- Keep PRs focused and small

## Community Standards

Be respectful. We follow the [Contributor Covenant](CODE_OF_CONDUCT.md).
