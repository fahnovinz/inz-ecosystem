# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| < 0.2   | No        |

## Reporting a vulnerability

If you discover a security issue, please report it responsibly:

1. **Do not** open a public issue
2. Email or DM [@fahnovinz](https://github.com/fahnovinz) with details
3. Include steps to reproduce if possible

We aim to respond within 7 days.

## Scope

INZ Ecosystem only fetches public data from the GitHub API. It does not store credentials, cache user data, or make requests to third-party services beyond GitHub and shields.io badge URLs.
