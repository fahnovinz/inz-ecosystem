# github-stats

Profile and portfolio analytics for any public GitHub user.

## CLI

```bash
node ../../bin/inz.js stats <username>
node ../../bin/inz.js stats <username> --json
```

## PowerShell

```powershell
../../scripts/github-stats.ps1 -Username <username>
```

## Data sources

All data comes from the public GitHub REST API. No data is stored or cached.
